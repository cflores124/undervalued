# Undervalued

Undervalued is a sports analytics platform focused on uncovering strategic patterns, statistical insights, and structural dynamics that shape sports outcomes.

The project combines analytical research with modern web development to present long-form sports analysis alongside interactive tools and visualizations.

## Live Website
https://undervaluedanalytics.com

## Features
- Long-form analytical articles
- Interactive modeling tools
- Insight cards for quick analysis
- RSS feed for updates
- Static site generation for performance

## Tech Stack
Next.js  
TypeScript  
TailwindCSS  
Framer Motion  

## Purpose
Undervalued serves as both a publishing platform and a portfolio of analytical work across multiple sports including baseball and mixed martial arts.
